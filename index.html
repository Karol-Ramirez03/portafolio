Tecnologias

JavaScript (JS) es un lenguaje de programación que permite agregar interactividad a las páginas web. Se ejecuta en el navegador y es esencial para crear funciones dinámicas como animaciones, validación de formularios, y la actualización del contenido sin recargar la página.

CSS (Cascading Style Sheets) es utilizado para describir la presentación visual de una página web. Permite definir el diseño, colores, fuentes, y disposición de los elementos HTML, mejorando la apariencia y el estilo de una página web.

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar páginas web. Define el contenido y la estructura de un sitio web mediante etiquetas, permitiendo agregar texto, imágenes, enlaces y otros elementos multimedia.
